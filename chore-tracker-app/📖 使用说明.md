# 🏠 家庭家务管理系统 - 使用说明

## 📁 文件夹清理说明

**请手动删除以下无关文件**（只保留 `index.html` 和 `start_server.py`）：

- ❌ 所有 `.bat` 文件（deploy.bat、启动.bat 等）
- ❌ 所有 `.ps1` 文件（deploy.ps1）
- ❌ 所有 `.md` 文档（README 等）
- ❌ 所有 `.txt` 文件
- ❌ `requirements.txt`
- ❌ `app.py`
- ❌ `钉钉集成版` 文件夹
- ❌ `cleanup.py`（清理脚本本身）

**最终只保留这 3 个文件：**
- ✅ `index.html` - 网页文件
- ✅ `start_server.py` - 服务器脚本
- ✅ `📖 使用说明.md` - 本说明文档

## 🚀 快速启动

### 方法 1：双击运行（推荐）

直接双击 **`start_server.py`** 文件即可启动服务器。

### 方法 2：命令行启动

1. 在当前文件夹空白处，按住 `Shift` + 鼠标右键
2. 选择"在此处打开 PowerShell 窗口"
3. 输入命令：
   ```powershell
   python start_server.py
   ```

## 📱 手机访问方法

启动服务器后，会显示两个地址：

```
📱 在以下设备上访问:
   • 本机访问：http://localhost:8080
   • 手机/其他设备：http://192.168.x.x:8080
```

**在手机上的操作步骤：**

1. 确保手机连接到**与电脑相同的 WiFi**
2. 打开手机浏览器（Safari/Chrome 等）
3. 输入显示的 IP 地址（如 `http://192.168.1.100:8080`）
4. 输入密码 `888888` 即可使用

## ❓ 常见问题

### Q: 手机无法访问？
A: 检查以下几点：
1. 手机和电脑是否连接**同一个 WiFi**
2. 防火墙是否阻止了 8080 端口
3. IP 地址是否正确

### Q: 端口被占用？
A: 修改 `start_server.py` 中的 `PORT = 8080` 为其他端口（如 8081）

### Q: 数据丢失了？
A: 可能是清除了浏览器缓存。建议定期截图备份重要统计数据。
